# Sriram Mudunuri Portfolio

Static personal portfolio site for GitHub Pages.

## Intended URL

This package is configured for:

https://legendarybladerofmilkywaygalaxy.github.io/

This version is configured for your current GitHub username and will publish at the URL above.


## Upload steps for GitHub Pages

1. Create a GitHub repository named exactly `legendarybladerofmilkywaygalaxy.github.io` under your GitHub account.
2. Upload all files from this folder to the root of the repository:
   - `index.html`
   - `styles.css`
   - `js/main.js`
   - `assets/`
3. Go to Repository Settings -> Pages.
4. Select `Deploy from a branch`.
5. Select branch `main`, folder `/root`.
6. Wait a few minutes, then open `https://legendarybladerofmilkywaygalaxy.github.io/`.
